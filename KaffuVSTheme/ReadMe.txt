## Use üöÄ

### Through the Visual Studio Marketplace üì¶

- Open the [Visual Studio Marketplace](https://marketplace.visualstudio.com/vs) and search for `Kaffu's Theme`.
- Click on the `Download` button.
- Go to `Tools > Options > Environment > General` and select `Kaffu Theme` in the `Color theme` dropdown.

### Through the VSIX File üì¶

- Download the [.vsix](https://github.com/Kaffu093/KaffuVSTheme/releases) from the releases section.
- Execute the `.vsix` file.
- Go to `Tools > Options > Environment > General` and select `Kaffu Theme` in the `Color theme` dropdown.

## Contributing ü§ù

If you think that this theme needs more languages or some changes, you can open an issue or a pull request.

If you want to open an issue, you can follow the next steps:

- Go to the [issues](https://github.com/Kaffu093/KaffuVSTheme/issues) section.
- Click on the `New issue` button.
- Fill the form with the information of the issue.
- Click on the `Submit new issue` button.

If you want to open a pull request, you can follow the next steps:

- Fork the project.
- Create a new branch (`git checkout -b feature/AmazingFeature`).
- Commit your changes (`git commit -m 'Add some AmazingFeature'`).
- Push the changes (`git push origin feature/AmazingFeature`).
- Open a Pull Request.

I'll be so grateful if you contribute to the project!
